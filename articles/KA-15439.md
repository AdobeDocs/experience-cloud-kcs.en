<b>Cause</b>
Some missing JAR files are reported by the WEB process.


## Resolution


<b>Solution</b>

The `classpath` tag from the `serverConf.xml` configuration file hasÂ not been updated correctly after the upgrade.

The `classpath` tag should point to the latest JARs.
